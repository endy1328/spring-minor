# Spring Miner 릴리즈 노트

## Version 1.0.0

### 개요

`Spring Miner`의 첫 번째 전달 가능 버전이다. Spring 기반 레거시 코드와 DB 자산의 명명 정보를 추출하고, 검색/조회/비교할 수 있는 기본 기능을 포함한다.

### 포함 기능

- 브라우저 기반 명명 정보 추출 UI
- 파일 업로드, 폴더 업로드, 코드 붙여넣기
- 스캔 결과 JSON 업로드
- 검색
- 결과 유형 필터
- 문서 유형 필터
- 상세 패널
- CSV 다운로드
- 스냅샷 저장 / 불러오기 / 삭제
- 현재 결과와 저장 스냅샷 비교
- 연관 항목 표시
- 로컬 프로젝트 디렉터리 스캔 CLI
- Java AST 기반 로컬 스캔 고도화

### 검증 항목

- `app.js` 문법 확인
- `scan-project.mjs` 문법 확인
- `JavaAstScanner.java` 컴파일 확인
- 샘플 스캔 실행 확인
- 브라우저 테스트 명령 실행 확인

### 산출물

- UI 실행 파일
- 로컬 스캔 CLI
- 샘플 데이터
- 테스트 스크립트
- 기획/운영/배포 문서

### 알려진 한계

- 브라우저 UI의 Java 분석은 정규식 기반이다.
- XML, SQL은 여전히 규칙 기반이다.
- 스냅샷 저장은 브라우저 `localStorage`에 의존한다.
- 연관 관계 표시는 `sourcePath` 공존 기준이다.

### 다음 확장 후보

- XML / SQL 파서 고도화
- 저장 스냅샷 영속화
- 스냅샷 간 직접 비교 강화
- 연관 관계 시각화 고도화
- 서버형 배포 구조
